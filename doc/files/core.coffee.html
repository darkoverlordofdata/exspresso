    + (Object) create_mixin($object, $args...)

    Create a Mixin

    Create a mixin object from a prototype object with an optional list of classes to mixin, followed by an optional list of arguments to the constructor of the first class.

    If there are no classes, the list of args is a list of additional objects that are simply merged into the first object. These objects are expected to be literal based, and only the own properties are used

    Parameters:

    • object (Object) — to use as the prototype
