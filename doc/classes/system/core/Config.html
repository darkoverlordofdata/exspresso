    - (Boolean) load($file = 'config', $use_sections = false, $fail_gracefully = false, $module = '')

    Load Config File

    Checks in MODPATH first. If not found, there, look the APPPATH and package folders.

    Parameters:

    • file (String) — the config file name
    • use_sections (Boolean) — if configuration values should be loaded into their own section
    • fail_gracefully (Boolean) — true if errors should just return false, false if an error message should be displayed
    • module (String) — module name from the uri

    Returns:

    • (Boolean) — if the file was loaded correctly

    - (Boolean) _application_load($file = '', $use_sections = false, $fail_gracefully = false)

    Load Application Config File
